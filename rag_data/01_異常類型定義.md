# 半導體製程異常類型定義手冊

## 1. ILD（Inter-Layer Dielectric）層異常

### 1.1 厚度偏薄（Thin ILD）
**定義**：ILD 膜厚低於規格下限（通常低於 target ± 5% 以下）
**典型原因**：
- PECVD 沉積時間縮短
- 前驅氣體（TEOS）流量異常偏低
- 腔體溫度偏低導致成長速率下降
- 晶圓放置位置偏移（非標準 boat position）

**標準判斷流程**：
1. 取 49 點厚度量測 map，確認均勻性（Uniformity < 2%）
2. 對比當班 PECVD log 確認 RF Power / Gas Flow
3. 若均勻性差 → 懷疑 shower head 堵塞
4. 若整片偏薄且均勻 → 懷疑氣體流量計（MFC）異常

**風險等級**：HIGH（影響金屬線間絕緣）
**對應處置**：Hold lot，通知 process engineer，執行 SEM 截面確認

---

### 1.2 粒子污染（Particle Contamination）
**定義**：晶圓表面出現 > 0.2μm 異常粒子，計數超標
**典型原因**：
- 腔體內壁沉積物剝落
- 機械臂接觸異常
- Clean room 環境不潔（HVAC 問題）
- 載具（FOUP）污染

**標準判斷流程**：
1. 確認粒子分佈（隨機 or 有規律 pattern）
2. Pattern 規律 → 機械臂或固定污染源
3. 隨機分佈 → 環境或腔體問題
4. 同一 lot 多片連續 → 設備問題（Hold 設備）
5. 單片 → 可能為 FOUP 或 cassette 問題

**風險等級**：MEDIUM~HIGH（視粒子位置而定）
**對應處置**：超標 → PM 清機，確認 clean 後放行

---

### 1.3 TiN（氮化鈦）阻障層異常
**定義**：TiN 薄膜電阻（Sheet Resistance）偏離規格
**典型原因**：
- PVD 靶材消耗接近 EOL（End of Life）
- Ar/N₂ 氣體比例漂移
- 基板溫度控制異常
- 腔體真空度不足

**標準判斷流程**：
1. 查看 Rs map 均勻性及趨勢
2. 確認靶材使用壽命（Target Life）
3. 對比同腔其他 lot 趨勢（是否有 drift）
4. Rs 整片偏高 → N₂ 比例過高（介電性增加）
5. Rs 整片偏低 → N₂ 比例不足（金屬性過強）

**風險等級**：MEDIUM（影響擴散阻障效果）
**對應處置**：Rs 偏離 ±8% → 調整 recipe，>±15% → 換靶材

---

## 2. STI（Shallow Trench Isolation）異常

### 2.1 CMP 研磨不均（Dishing / Erosion）
**定義**：CMP 後 STI 區域出現下凹（Dishing）或周邊氧化層過度研磨（Erosion）
**典型原因**：
- 研磨墊使用時間過長（Pad Age）
- 漿料（Slurry）流量異常
- 研磨頭壓力不均
- 終點偵測（EP detection）失靈

**風險等級**：HIGH（影響後續 gate 製程平坦度）

---

### 2.2 Void 形成
**定義**：STI 填充氧化層內出現空洞
**典型原因**：
- HDP-CVD 參數異常
- 溝槽（trench）高寬比（AR）過高
- Bottom-up fill 速率不足

**風險等級**：CRITICAL（可能造成漏電或元件失效）

---

## 3. BPSG（硼磷矽玻璃）層異常

### 3.1 回流不完全（Incomplete Reflow）
**定義**：BPSG 高溫回流後平坦化效果不足，殘留台階高度超標
**典型原因**：
- 回流溫度不足（< 850°C）
- B/P 濃度偏低（降低玻璃轉化溫度不足）
- 氮氣環境不純（含微量氧）

**風險等級**：MEDIUM（影響後續金屬填孔 step coverage）

---

## 4. Metal（金屬層）異常

### 4.1 W-plug 填充不完全（Tungsten Void）
**定義**：鎢栓塞（W contact/via plug）內部出現空洞
**典型原因**：
- TiN barrier layer 不連續
- WF₆ 前驅氣體分解不均勻
- CVD 腔體溫度分佈不均

**風險等級**：CRITICAL（接觸電阻異常升高）

### 4.2 Al 金屬線電遷移風險（EM Risk）
**定義**：鋁金屬線在高電流密度下發生原子遷移，導致斷線或 hillock
**典型原因**：
- 金屬線寬低於設計規則最小值
- 晶格結構（grain size）不均勻
- TiN/Ti capping layer 缺失

**風險等級**：HIGH（長期可靠性問題）
