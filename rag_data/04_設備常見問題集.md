# 設備常見問題與故障排除知識庫

## PECVD（電漿增強化學氣相沉積）

### 常見問題 Q&A

**Q：PECVD 腔體壓力飄高，可能原因是什麼？**
A：常見原因包括：
1. 排氣幫浦（Pump）效能下降 → 檢查 pump 油位及 foreline pressure
2. 氣體管路洩漏（leak） → 執行 leak check
3. Throttle valve 控制器異常 → 確認 APC（Automatic Pressure Control）閥位
4. Process gas 流量突然增加 → 確認 MFC 是否有 spike

**Q：SiO₂ 沉積速率突然下降 15%，最可能原因？**
A：優先排查順序：
1. RF Power 實際輸出下降（confirm via RF log）
2. TEOS 液態前驅氣體槽體液位偏低（蒸氣壓下降）
3. Shower head 孔洞部分堵塞（不均勻沉積分佈）
4. 腔體清潔週期已到（thin film 剝落影響成長環境）
處置：執行 NF₃ 腔體清潔後確認速率恢復

**Q：晶圓表面出現圓形同心圓粒子分佈，意味著什麼？**
A：同心圓分佈強烈暗示 shower head 問題：
- 中心孔洞堵塞 → 中心粒子多，邊緣少
- 邊緣孔洞堵塞 → 邊緣粒子多
- Shower head 本體有裂縫 → 不規則同心圓
建議立即 Hold 設備，執行 shower head 更換

## PVD（物理氣相沉積）

### 常見問題 Q&A

**Q：TiN 片電阻（Rs）持續單向漂移，如何判斷靶材壽命？**
A：
1. 查看靶材使用紀錄（kWh 累積）
2. 確認靶材使用率（通常 > 80% 即需更換）
3. 觀察 Rs 趨勢：若呈現線性升高 → 靶材侵蝕導致 composition 改變
4. 執行靶材厚度量測（若有 in-situ 監控）
判斷：若靶材使用率 > 85% + Rs 漂移 > 5% → 建議計劃換靶

**Q：PVD 濺鍍後晶圓中心與邊緣 Rs 差異 > 3%，原因？**
A：均勻性問題主要來自：
1. 磁控管（Magnetron）磁場不對稱
2. 靶材侵蝕不均（edge erosion vs center erosion）
3. 基板旋轉機構故障（未均勻旋轉）
4. 基板溫度分佈不均

## CMP（化學機械研磨）

### 常見問題 Q&A

**Q：CMP 後量測發現 WIWNU（Within-Wafer Non-Uniformity）突然變差，如何排查？**
A：
1. 確認研磨墊（Pad）使用時數是否接近 EOL
2. 確認 Conditioner 鑽石盤使用壽命（關鍵！）
3. 查看 Slurry 流量及溫度記錄（漿料溫度影響速率）
4. 確認研磨頭各 zone 壓力設定是否有變動
5. 若邊緣特別差 → 懷疑 retaining ring 磨損

**Q：ILD CMP 出現 Dishing，如何確認及處置？**
A：
- Dishing 定義：大面積金屬或介電層表面下凹
- 量測方法：AFM 或 Profilometer 量測凹陷深度
- 原因：研磨速率 > 填充材料硬度，過度研磨
- 處置：縮短研磨時間、降低壓力，或更換低-dishing slurry formula

## 蝕刻（Dry Etch）

### 常見問題 Q&A

**Q：Gate etch 後 CD 偏小（under-etch 方向），可能原因？**
A：
1. 光阻（Photoresist）CD 本身已偏小 → 問題在 litho，非 etch
2. Etch time 過短或 RF power 偏低
3. 終點偵測（End Point Detection）提前觸發
4. Process gas 比例（Cl₂/HBr 比）異常
排查：確認 EPD 信號是否正常，比對 resist CD 量測值

**Q：Oxide etch 出現 micro-loading effect，如何處理？**
A：
- 現象：密集圖案區域蝕刻速率低於稀疏區域
- 原因：局部 reactive species 耗盡
- 處置：
  1. 降低 pressure（減少 mean free path）
  2. 增加 ion bombardment（提高 bias power）
  3. 調整 gas composition
  4. 考慮 pattern density 補償（光罩修正）

## MES 系統常見操作問題

**Q：MES 中 Hold lot 後發現 hold 錯了，如何解除？**
A：
1. 確認你有 Hold Release 權限（需 Process Engineer 或以上）
2. 在 Lot Management → Hold List 找到該 lot
3. 選擇 Release，填寫 Release Reason（必填）
4. 若 Hold 涉及設備 OOC，需同步確認設備已回到 in-control 狀態
5. 無法自行解除 → 聯繫 Section Manager 審核

**Q：MES 系統中如何查詢某設備最近一個月的 OOC 記錄？**
A：
1. 進入 SPC Module → Equipment 選擇對應設備
2. 時間範圍選擇最近 30 天
3. 選擇 "OOC Events" filter
4. 可匯出 CSV 進行進一步分析
5. 若需要跨設備比較，使用 Report → Equipment Performance Report

**Q：AI Copilot 分析出現 "evidence_source: memory"，這個記憶是從哪來的？**
A：
- 記憶來自 `ai_memory_events` 集合，儲存歷史分析結果
- 每次 AI 成功分析後自動寫入
- 檢索時按照 layer / machine / anomaly_type / 時間近度 排序
- "memory" 代表 AI 找到歷史相似案例並納入判斷，通常比純 "live" 分析更準確
